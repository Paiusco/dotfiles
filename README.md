# My dotfiles

This repo contains [@paiusco](https://github.com/Paiusco) dotfiles for:

- Fish
- Zsh
- Neovim
- Alacritty
- Ranger
- i3wm
- Qtile


## Usage

    git clone https://github.com/Paiusco/dotfiles.git ~/dotfiles
    cd ~/dotfiles
    ./bootstrap.sh
